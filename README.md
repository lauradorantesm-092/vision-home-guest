## Deploy to Cloudflare Pages (Direct Upload)
Cloudflare supports direct uploads (drag & drop or Wrangler). See official docs:
- Direct Upload: https://developers.cloudflare.com/pages/get-started/direct-upload/

Tip: If you choose Git integration, you can't switch to Direct Upload later.
